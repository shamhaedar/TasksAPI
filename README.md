# TasksAPI 🚀

A REST API for managing tasks built with ASP.NET Core .NET 10

## Features
- Full CRUD operations (Create, Read, Update, Delete)
- RESTful API design
- Swagger UI documentation
- Professional error handling

## Tech Stack
- ASP.NET Core .NET 10
- C#
- Swagger/OpenAPI

## Endpoints

| Method | Endpoint | What it does |
|--------|----------|-------------|
| GET | `/api/Tasks` | Get all tasks |
| GET | `/api/Tasks/{id}` | Get one task by ID |
| POST | `/api/Tasks` | Create new task |
| PUT | `/api/Tasks/{id}` | Update a task |
| DELETE | `/api/Tasks/{id}` | Delete a task |

## How to Run
1. Clone: `git clone https://github.com/shamhaedar/TasksAPI.git`
2. Open folder: `cd TasksAPI`
3. Run: `dotnet run`
4. Open browser: `http://localhost:5230/swagger`
